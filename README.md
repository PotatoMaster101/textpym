# Text Pyramid
Generates a text pyramid in C++14.

# Compile
Use C++14 or higher:
```
$ g++ -std=c++14 -Wall -Wextra src/*.cpp -o pym
```

# Usage
Invoke the program with n number of arguments:
```
$ ./pym abcdef 123456
a
ab
abc
abcd
abcde
abcdef
abcdef 1
abcdef 12
abcdef 123
abcdef 1234
abcdef 12345
abcdef 123456
abcdef 12345

```
